ways to alter font-size

absolute, relative, unit, viewport, percentage

when specifying a percentage font-size, percentage is of the parent element. this font-size is 75% of the parent element

em

an "em" is a font-size relative to a parent element size. 4em is 4*the size of the parent element. this is 5em.

rem

rem--relative to the root html element font size, not the parent html element.you can change the root html font-size by assigning a value to it in css /html{font-size:...}


it's useful to set the html element to a viewport amount, and the other elements to a rem of the root element. when the browser viewport changes, everything else will change proportionally--but be careful if they virtually disappear in a small viewport
