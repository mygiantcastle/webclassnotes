css syntax

every css file no matter the complexity will follow the same syntactical pattern


select-some-html-element { property: value; }
e.g., body { background-color: purple; }
    elements of css syntax
  • selector--"some-html-element"
  • declaration--the content inside of the curly braces
  • property--the "what" of the selector to be modified (left of the declaration)
  • here is link to a reference listof css properties
  • value--the "how" of the modification (right of the declaration)
