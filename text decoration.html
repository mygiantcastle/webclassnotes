text decoration allows us to set an underline or an overline, line-through, dotted, dash, etc.. for reference

you can use text decoration in a nav bar to remove the underline to hyperlinks such as in the below example

home contact

line-throughs are useful when creating apps with to-do lists for users to cross of items...

the text-decoration property contains several other properties that can be manipulated, such as style, color, thickness, line...

be careful of browser compatibility, however.

order of values to the text-decoration element is not significant
