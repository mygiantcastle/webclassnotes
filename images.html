There is an element called "img." There is no closing tag, only an opening tag. There is no text content, just an image (png, gif, jpeg).

IMG needs an attribute. something=something else. SRC (=source) = image.file.

Although HTML allows images to be modified by size and width, CSS is more often used to do this task.

Essential information should not be contianed in an image because a user's browser may not support it, or the image may not appear because the server or address may be down.

Screen readers will read the alt attribute aloud if the image fails to load.
