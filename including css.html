where do we write our css?
  • in codepen.com, css can be written in a window adjacent to an html window
      • three ways to write css
    1. in our html file using a /style/ element
    2. external css file using a /link/ element -- preferable choice
    3. inline (don't do this)

inline style

less than ideal because each anchor tag to be styled must be done individually

inside our html file using a /style/ element in the /head

reduces need to repeat style elements for several tags.

external style sheet (css file)

can be used across several html documents
to include a css file in a html file, must use a link element in the /head element.
/link rel="stylesheet" href="including css.css"/
