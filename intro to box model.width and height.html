what is the box model?

every element in html has a box around it, whether visible or not.
each box consists of multiple properties that control the box--padding, border,

elements of a box in html

  1. content
  2. padding
  3. border
  4. margin

to alter the width/height of the content

simply specifiy the content in css /element {width:...; height:...;}


when applying a value of percent for box size, the percent amount is a percentage of the width(or height) of its parent box
can adjust min-height, min-width, max-height, max-width.
