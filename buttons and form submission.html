by entering information into a form and hitting enter, the form automatically submits unless the type is of a certain value

one method is to create an input element with a type attribute equal to "submit"

using the submit type in html requires a textual value. if you desire to create a submit form with content that is nontextual like an image, you must not use html for that purpose

you can also use a button element to submit a form

if you have 2 buttons within the same "form" element, either button will submit the form, regardless of where the button in the form is located. this will cause problems.

the default type attribute of button element is "submit". That type can be altered. button type="button" will not submit the form.

button allows more content besides text than submit elements. use button instead.
