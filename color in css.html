What are the possible values for a given property="color" in css?

  • previously defined colors (=147)
  • RGB colors (=16.7 million)
    ...color: rgb(red0-255,green0-255,blue0-255)
  • RGBA colors
    red, green, blue, and alpha channels. alpha=transparency value
  • HEX (hexadecimal="base 16") =16.77million colors
    #------
  • HSL = hue, saturation, and lightness
    hub=0-360, saturation=0%-100%, lightness=0%-100%
    hsl is valuable because you can use the same hue for background and text but adjust the saturation and/or lightness, or find a complementary hue, etc...
    what if we wanted to cast a shadow on a background? we would use an hsl color and adjust the lightness value.
