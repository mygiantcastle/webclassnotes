important attributes to know:
  • minlength--min number of chars to be valid
  • maxlength--max num chars to be valid
  • placeholder--description that appears when input is empty
  • required--whether the input is required

by entering an input into a form and hitting "enter," the user successfully submits the form. the html page is automatically reloaded and the url is altered to read ..../form name=userinput

validation of a form input is used with javascript typically

required attribute is boolean attribute
